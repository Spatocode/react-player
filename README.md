# React Player

Audio player web application with react.


## Teck Stack

* React
* Redux

## Features

* File API
* Song rating
* Play progress bar
* Song play/pause/stop
* Song fast-forward/rewind

Feel free to contribute to this repo. Every contributions are greatly appreciated.
